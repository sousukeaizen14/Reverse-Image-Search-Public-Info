# Security Policy

## Overview
This project is designed with privacy and safety as a priority.
It performs reverse image search to retrieve **publicly available information**
from the web and does NOT attempt to identify individuals.

---

## Data Handling
- Uploaded images are processed temporarily in memory
- Images are deleted immediately after processing
- No images are stored, logged, or cached
- No personal data is collected or tracked

---

## Identification Policy
- This project does NOT perform face recognition
- This project does NOT identify or confirm a person's identity
- All results are sourced from public web pages
- Interpretation of results is the responsibility of the user

---

## API Security
- API keys must be stored using environment variables
- API keys should never be committed to the repository
